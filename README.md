## 📌 Sobre o projeto

Este projeto é uma **réplica da landing page institucional do Discord** (site de marketing, não o aplicativo interno). Foi desenvolvido como parte dos desafios de CSS da [DIO](https://dio.me), com foco em:

- **Layout responsivo** (mobile, tablet, desktop)
- **Acessibilidade** (skip-link, semântica HTML5, contraste)
- **CSS moderno** (Flexbox, variáveis, media queries)
- **Boas práticas** de organização e performance

A página apresenta os principais recursos do Discord, chamadas para ação, seção de tecnologia e rodapé completo, tudo com identidade visual semelhante à oficial.
